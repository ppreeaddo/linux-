# Linux System Administration & Ansible Automation
# Description
A hands on Red Hat Enterprise Linux project focused on automating system administration tasks using Ansible, including user management, file management, and enforcing access controls across multiple managed hosts.
# Tools Used
Red Hat Enterprise Linux, Ansible & Ansible Navigator, YAML Playbooks, SSH, Linux CLI
# What I Did
Wrote and executed Ansible playbooks to manage user accounts across multiple remote servers. Configured SSH authentication and tested access controls including root login restrictions. Automated file and directory management across managed hosts using Ansible modules. Verified successful task execution using Ansible play recap output.
# What I Learned
How to write and run Ansible playbooks using YAML, manage users and enforce authentication policies on remote Linux systems and automate system administration tasks across multiple hosts.
